# ABC Pharmacy App

## Run Backend

```bash
cd PharmacyApi
dotnet run --urls=http://localhost:5000
```

Open:

```
http://localhost:5000/api/medicine
```

---

## Run Frontend

```bash
cd pharmacy-ui
npm install
npm run dev
```

Open:

```
http://localhost:5173
```

---

## Notes

* Backend runs on: http://localhost:5000
* Frontend runs on: http://localhost:5173
* Data is stored in `data.json`
